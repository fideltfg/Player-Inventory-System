Item Class

Defines the basic properties and methods of an item.

Public Fields

Data

This item's catalog entry is stored here.

public ItemData Data;

StackCount

How many copies of this item are currently stacked here.

public int StackCount { get; protected set; }

Durability

The current durability value of this item.

public float Durability = 0;

Stackable

Indicates if the item can be stacked.

public bool Stackable { get; }

lastSlot

The last slot this item was in.

public Slot lastSlot;

Public Methods

New

Creates a new Item of the given ID with the given stack count.

public static Item New(int itemID, int count = 1);

Parameters:

  • itemID: The ID of the Item to return.
  • count: The stack count (default is 1).

Returns:

  • Item: An Item with the specified stack count, or null if invalid item ID.

AddToStack

Adds a quantity to this item's stack.

public bool AddToStack(int quantity);

Parameters:

  • quantity: The quantity to add to the stack.

Returns:

  • bool: True if the quantity was successfully added, false otherwise.

SetStackCount

Sets the value of StackCount.

public bool SetStackCount(int val);

Parameters:

  • val: The value to set.

Returns:

  • bool: True if the value was successfully set, false otherwise.

Clone

Clones this Item.

public Item Clone();

Returns:

  • Item: A memberwise clone of this Item.

Uses

Contains a list of items that need this item to be crafted.

public string[] Uses { get; }

Returns:

  • string[]: An array of item names that require this item to be crafted.

Place

Places an item in the world and removes it from the inventory.

internal static void Place(Item item, Vector3 position, Quaternion rotation, Vector3 scale);

Parameters:

  • item: The item to place.
  • position: The position to place the item.
  • rotation: The rotation to place the item.
  • scale: The scale to place the item.

Usage Examples

Example: Create a new item

Item newItem = Item.New(1, 5);
if (newItem != null) {
    Debug.Log("Item created: " + newItem.Data.name);
} else {
    Debug.Log("Failed to create item");
}

Example: Add to an item's stack

bool success = newItem.AddToStack(3);
if (success) {
    Debug.Log("Added to stack. New stack count: " + newItem.StackCount);
} else {
    Debug.Log("Failed to add to stack");
}

Example: Set an item's stack count

bool success = newItem.SetStackCount(10);
if (success) {
    Debug.Log("Stack count set. New stack count: " + newItem.StackCount);
} else {
    Debug.Log("Failed to set stack count");
}

Example: Clone an item

Item clonedItem = newItem.Clone();
Debug.Log("Cloned item: " + clonedItem.Data.name);

Example: Get items that use this item in their recipe

string[] uses = newItem.Uses;
Debug.Log("Items that use this item in their recipe: " + string.Join(", ", uses));

Example: Place an item in the world

Item.Place(newItem, new Vector3(0, 0, 0), Quaternion.identity, Vector3.one);
Debug.Log("Item placed in the world");
