Inventory Class

Provides methods to store, modify, and search items in the inventory system.

Public Fields

Index

The index of this inventory in the InventoryController InventoryList, set at object creation.

public int Index { get; protected set; }

Constructors

Inventory

New Inventory Constructor.

internal Inventory(int index, int capacity);

Parameters:

  • index: The index that this inventory will reside in the InventoryList of InventoryController.
  • capacity: The number of slots available in this inventory.

Inventory

Inventory Constructor from Serial Inventory.

internal Inventory(SerialInventory sInventory);

Parameters:

  • sInventory: The serial inventory from which to create this inventory.

Public Methods

InventoryContains

Method to check if this inventory contains an item.

public bool InventoryContains(Item item);

Parameters:

  • item: The item to check for in the inventory.

Returns:

  • bool: True if the item is found in the inventory, false otherwise.

InventoryContains

Method to check if this inventory contains an item with the given ID.

public bool InventoryContains(int itemID);

Parameters:

  • itemID: The ID of the item to check for in the inventory.

Returns:

  • bool: True if the item is found in the inventory, false otherwise.

GetItemSlots

Method to return all the slots in the inventory that contain the given item.

public List GetItemSlots(Item item);

Parameters:

  • item: The item to search for in the inventory.

Returns:

  • List<Slot>: A list of slots containing the given item.

PlaceItemInNewSlot

Method to place an item into an empty slot of this inventory. If there are no empty slots, this method will return false.

public bool PlaceItemInNewSlot(Item item);

Parameters:

  • item: The item to be placed in the inventory.

Returns:

  • bool: True if the item is successfully placed, false otherwise.

AddItem

Method to add an item to this inventory. The item will be added to the first existing slot that already contains an instance of the given item (stacked) or if none exist it will be placed in an empty slot. If there are no empty slots, this method will return false.

public bool AddItem(Item item);

Parameters:

  • item: The item to be added.

Returns:

  • bool: True if the item is successfully added, false otherwise.

EmptySlotCount

Property to return the number of empty slots available in this inventory.

public int EmptySlotCount { get; }

Returns:

  • int: The number of empty slots.

GetSlotsWithItem

Method to return an array of all the slots in this inventory that contain the item held in the given slot. If the given slot is from this inventory it will be excluded from the array.

public Slot[] GetSlotsWithItem(Slot slot);

Parameters:

  • slot: The slot containing the item to search for.

Returns:

  • Slot[]: An array of slots containing the same item.

GetItemCount

Method to return the count of the given item in this inventory.

internal int GetItemCount(int itemID);

Parameters:

  • itemID: The ID of the item to count.

Returns:

  • int: The count of the item in the inventory.

EmptyAllSlots

Method to empty all slots in this inventory.

internal void EmptyAllSlots();

Usage Examples

Example: Add an item to the inventory

Item newItem = Item.New(1, 5);
bool added = inventory.AddItem(newItem);
if (added) {
    Debug.Log("Item added to inventory");
} else {
    Debug.Log("Failed to add item to inventory");
}

Example: Check if inventory contains an item

bool containsItem = inventory.InventoryContains(newItem);
if (containsItem) {
    Debug.Log("Item found in inventory");
} else {
    Debug.Log("Item not found in inventory");
}

Example: Get the count of a specific item in the inventory

int itemCount = inventory.GetItemCount(1);
Debug.Log("Item count: " + itemCount);

Example: Get all slots containing a specific item

Slot[] slotsWithItem = inventory.GetSlotsWithItem(someSlot);
if (slotsWithItem != null) {
    foreach (Slot slot in slotsWithItem) {
        Debug.Log("Slot ID: " + slot.SlotID + ", Stack Count: " + slot.StackCount);
    }
} else {
    Debug.Log("No slots contain the item");
}

Example: Empty all slots in the inventory

inventory.EmptyAllSlots();
Debug.Log("All slots emptied");
