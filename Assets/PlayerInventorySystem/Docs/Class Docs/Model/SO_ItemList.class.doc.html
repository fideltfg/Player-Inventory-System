SO_ItemList Class

A ScriptableObject used to create the Item Catalog.

Public Fields

catalogID

The ID of the item catalog.

public int catalogID = 0;

list

A list of ItemData objects representing the items in the catalog.

public List list = new List();

Public Methods

GetItemByID

Gets an item by its ID.

public Item GetItemByID(int itemID);

Parameters:

  • itemID: The ID of the item to retrieve.

Returns:

  • Item: The item with the specified ID.

GetRandomItem

Gets a random item from the catalog.

public Item GetRandomItem();

Returns:

  • Item: A random item from the catalog.

Usage Examples

Example: Get an item by its ID

int itemID = 1;
SO_ItemList itemList = ScriptableObject.CreateInstance<SO_ItemList>();
Item item = itemList.GetItemByID(itemID);
Debug.Log("Retrieved item: " + item.Name);

Example: Get a random item from the catalog

SO_ItemList itemList = ScriptableObject.CreateInstance<SO_ItemList>();
Item randomItem = itemList.GetRandomItem();
Debug.Log("Random item: " + randomItem.Name);
