Recipe Class

Defines and holds a crafting recipe.

Public Fields

rows

An array of Row objects representing the rows in the recipe.

public Row[] rows = new Row[3];

Public Properties

Ingredients

Gets the recipe in a format that can be used by the crafting system to test if a recipe is met.

public string Ingredients { get; }

Public Methods

InRecipe

Checks if an item is required for this recipe.

public bool InRecipe(int x);

Parameters:

  • x: The item ID to look for in this recipe.

Returns:

  • bool: True if the item is required, false otherwise.

GetRequiredItemIds

Gets an array of item IDs required for this recipe.

internal int[] GetRequiredItemIds();

Returns:

  • int[]: An array of item IDs required for this recipe.

Usage Examples

Example: Check if an item is in the recipe

Recipe recipe = new Recipe();
bool isInRecipe = recipe.InRecipe(1);
if (isInRecipe) {
    Debug.Log("Item is in the recipe");
} else {
    Debug.Log("Item is not in the recipe");
}

Example: Get the ingredients of the recipe

Recipe recipe = new Recipe();
string ingredients = recipe.Ingredients;
Debug.Log("Recipe ingredients: " + ingredients);

Example: Get required item IDs for the recipe

Recipe recipe = new Recipe();
int[] requiredItemIds = recipe.GetRequiredItemIds();
Debug.Log("Required item IDs: " + string.Join(", ", requiredItemIds));
