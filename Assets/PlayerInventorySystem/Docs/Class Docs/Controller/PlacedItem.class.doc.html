PlacedItem Class

Component added to item prefabs so the inventory system can identify them for saving and respawning.

Public Fields

ItemID

The ID of the item.

public int ItemID = 0;

Usage Examples

Example: Assign an ItemID to a placed item

GameObject placedItemObject = ...;
PlacedItem placedItem = placedItemObject.GetComponent<PlacedItem>();
placedItem.ItemID = 1;
Debug.Log("Placed item ID set to: " + placedItem.ItemID);
