SlotController Class

The base slot controller contains all methods for dealing with slots and provides access to items within the slot.

Public Fields

slotID

The ID of this slot.

public int slotID;

counter

Counter for internal timing.

public float counter = 0;

unselectedColor

The color of the slot when it is not selected.

public Color unselectedColor;

mouseOver

Indicates if the mouse is over the slot.

public bool mouseOver;

showInfoPanel

Indicates if the info panel should be shown.

public bool showInfoPanel;

interactable

Indicates if the slot is interactable.

public bool interactable = true;

infoPanel

The transform of the info panel.

public Transform infoPanel;

Index

The index of this slot's inventory.

public int Index = 0;

image

The UI image that this slot will display an item's sprite.

public Image image;

text

The text object that this controller will use to display the stack count.

public Text text;

dSlider

The slot durability slider.

public Slider dSlider;

outline

The outline component of the slot. Used to highlight the slot.

public Outline outline;

slotLocation

The location of this controller's slot.

public SLOTLOCATION slotLocation = SLOTLOCATION.INVENTORY;

HighlightColor

The default color for highlighting this slot.

public Color HighlightColor;

ValidColor

Highlight color when item placement is valid.

public Color ValidColor;

ErrorColor

The highlight color when item placement is invalid.

public Color ErrorColor;

Usage Examples

Example: Set a slot for the SlotController to control

Slot slot = new Slot(0);
SlotController slotController = GetComponent<SlotController>();
slotController.SetSlot(slot);
Debug.Log("Slot set for SlotController");

Example: Update the slot's UI

slotController.UpdateSlotUI();
Debug.Log("Slot UI updated");

Example: Handle pointer enter event

public void OnPointerEnter(PointerEventData eventData) {
    slotController.OnPointerEnter(eventData);
    Debug.Log("Pointer entered slot");
}

Example: Swap items between slots

slotController.SwapItems();
Debug.Log("Items swapped between slots");
