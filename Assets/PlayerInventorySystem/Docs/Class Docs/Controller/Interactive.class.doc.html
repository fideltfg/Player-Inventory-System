Interactive Class

Represents an interactive item that the player can interact with, which displays an inventory panel when triggered.

Public Fields

Panel

The panel the player will see when they interact with this object.

public InventorySystemPanel Panel;

Radius

The interaction radius within which the player can interact with this object.

internal int Radius = 2;

Public Methods

Update

Checks the distance between the interactive item and the player to determine if the panel should be closed.

public virtual void Update();

ClosePanel

Closes the interaction panel.

internal virtual void ClosePanel();

OpenPanel

Opens the interaction panel.

internal virtual void OpenPanel();

Usage Examples

Example: Open the interaction panel

Interactive interactive = GetComponent<Interactive>();
interactive.OpenPanel();
Debug.Log("Interaction panel opened");

Example: Close the interaction panel

interactive.ClosePanel();
Debug.Log("Interaction panel closed");

Example: Update the interactive item

interactive.Update();
Debug.Log("Interactive item updated");
