SalvageOutputSlotController Class

Handles interactions with slots in the salvage output panel, extending the functionality of the base `SlotController`.

Overrides

UpdateSlotUI

Updates the slot's UI and sets the outline color based on the salvageability of the item.

internal override void UpdateSlotUI();

Usage Examples

Example: Update the slot's UI in a salvage output slot

salvageOutputSlotController.UpdateSlotUI();
Debug.Log("Salvage output slot UI updated");
