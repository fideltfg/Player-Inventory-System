SalvageSlotController Class

Handles interactions with slots in the salvage panel, extending the functionality of the base `SlotController`.

Overrides

SetOutLineColor

Sets the outline color of the slot based on the recyclability of the held item.

internal override void SetOutLineColor();

OnPointerDown

Handles the mouse pointer down event on this slot.

public override void OnPointerDown(PointerEventData eventData);

Parameters:

  • eventData: The pointer event data.

Usage Examples

Example: Handle pointer down event on a salvage slot

public void OnPointerDown(PointerEventData eventData) {
    salvageSlotController.OnPointerDown(eventData);
    Debug.Log("Pointer down on salvage slot");
}

Example: Set the outline color of a salvage slot

salvageSlotController.SetOutLineColor();
Debug.Log("Outline color set for salvage slot");
