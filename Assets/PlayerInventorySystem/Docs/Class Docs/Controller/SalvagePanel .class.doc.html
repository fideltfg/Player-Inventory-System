SalvagePanel Class

Controller for the Salvage panel, handling the generation of the panel's slot list and cleanup when the panel is closed.

Public Fields

inputSlotController

Controller for the input slot.

public SalvageSlotController inputSlotController;

OutputArray

Transform for the output array.

public Transform OutputArray;

Protected Fields

validSalvage

Indicates if the current recipe is valid for salvage.

private bool validSalvage = false;

salvagableItemIDs

Array of item IDs that are salvageable.

private int[] salvagableItemIDs;

Public Methods

OnDisable

Called when the panel is disabled, moves remaining items back to the inventory or drops them on the ground, and clears the output slots.

public override void OnDisable();

Build

Builds the salvage panel.

public override void Build(int inventoryIndex);

Parameters:

  • inventoryIndex: Index of the inventory.

SlotChangeCallback

Callback for when the slot changes.

public void SlotChangeCallback(Slot slot);

Parameters:

  • slot: The changed slot.

SalvageButton

Handles the salvage button click, processing the output slots if the salvage is valid.

public void SalvageButton();

Usage Examples

Example: Build the salvage panel

SalvagePanel salvagePanel = new SalvagePanel();
salvagePanel.Build(0);
Debug.Log("Salvage panel built");

Example: Handle slot change in salvage panel

Slot slot = new Slot(0);
salvagePanel.SlotChangeCallback(slot);
Debug.Log("Slot change handled");

Example: Handle salvage button click

salvagePanel.SalvageButton();
Debug.Log("Salvage button clicked");
