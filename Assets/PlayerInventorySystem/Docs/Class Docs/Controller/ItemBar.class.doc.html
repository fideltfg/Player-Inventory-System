ItemBar Class

ItemBar Component Class

Public Fields

built

Indicates whether the item bar has been built.

public bool built = false;

Public Methods

OnEnable

Called when the item bar is enabled.

public override void OnEnable();

Update

Called once per frame to update the item bar, handling mouse wheel input for slot selection.

public override void Update();

Build

Builds the item bar with slots from the specified inventory index.

public override void Build(int InventoryIndex);

Parameters:

  • InventoryIndex: The index of the inventory to build slots from.

DropSelectedItem

Drops the item in the currently selected slot.

public void DropSelectedItem();

SelectNextSlot

Selects the next slot in the item bar.

public void SelectNextSlot();

SelectPreviousSlot

Selects the previous slot in the item bar.

public void SelectPreviousSlot();

GetSelectedItem

Gets the currently selected item (or stack of items) from the item bar without removing it.

public Item GetSelectedItem();

Returns:

  • Item: The currently selected item, or null if no item is selected.

ConsumeSelectedItem

Consumes the currently selected item, decrementing its stack count or removing it if the stack is empty.

public void ConsumeSelectedItem();

Usage Examples

Example: Build the item bar

ItemBar itemBar = GetComponent<ItemBar>();
itemBar.Build(0);
Debug.Log("Item bar built");

Example: Drop the selected item

itemBar.DropSelectedItem();
Debug.Log("Selected item dropped");

Example: Select the next slot

itemBar.SelectNextSlot();
Debug.Log("Next slot selected");

Example: Get the selected item

Item selectedItem = itemBar.GetSelectedItem();
if (selectedItem != null) {
    Debug.Log("Selected item: " + selectedItem.Data.name);
} else {
    Debug.Log("No item selected");
}

Example: Consume the selected item

itemBar.ConsumeSelectedItem();
Debug.Log("Selected item consumed");
