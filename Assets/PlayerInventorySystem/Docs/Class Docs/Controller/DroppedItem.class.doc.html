DroppedItem Class

Component to add to items that are dropped in the world to allow the player to pick them up. This component will change the object's tag to "Item" when used.

Public Fields

ItemID

The ID of the item the GameObject represents.

public int ItemID = 0;

StackCount

The stack count for the item.

public int StackCount = 1;

Durability

The current durability value for this item. Does not apply if the stack count > 1.

public float Durability = 0;

TimeToLive

The amount of time this object will remain after being dropped before despawning.

public float TimeToLive = 30;

Timer

Timer to track the elapsed time since the item was dropped.

public float Timer = 0;

Position

Holder for position data. Used only when the game loads saved inventory data; should be null otherwise.

public Vector3 Position;
