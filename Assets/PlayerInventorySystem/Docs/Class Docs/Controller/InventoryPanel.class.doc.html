InventoryPanel Class

Controller for the player inventory panel.

Public Methods

Build

Builds the inventory panel with slots from the specified inventory index.

public override void Build(int InventoryIndex);

Parameters:

  • InventoryIndex: The index of the inventory to build slots from.

Usage Examples

Example: Build the inventory panel

InventoryPanel inventoryPanel = GetComponent<InventoryPanel>();
inventoryPanel.Build(0);
Debug.Log("Inventory panel built");
