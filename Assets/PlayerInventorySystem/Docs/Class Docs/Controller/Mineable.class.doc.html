Mineable Class

Represents a mineable resource in the game that players can mine to obtain items.

Public Fields

resourceCount

The maximum number of items that can be mined from this resource.

[Tooltip("The number of items that could be mined here")]
public int resourceCount = 10;

hitRate

The number of hits it takes to mine one resource.

[Tooltip("How many hits it takes to mine one resource")]
public float hitRate = 10;

minedItemID

The ID of the item that will be mined from this resource.

[Tooltip("The item that will be mined from this mine by id")]
public int minedItemID = 0;

refreshRate

The amount of time it takes to reset the mining on this item and allow another hit.

[Tooltip("How long it takes for hit counter to reset to 0 should the player leave")]
public float refreshRate = 5;

Private Fields

hitCounter

The current number of hits on this item since the last refresh.

int hitCounter = 0;

refreshCounter

Counter to track the refresh time.

float refreshCounter = 0;

Public Methods

Mine

Mines this item. A hit is the act of the player striking the mineable resource once. This is time-limited by the refreshRate.

internal Item Mine(float hitPower);

Parameters:

  • hitPower: How hard the player hits the mineable resource.

Returns:

  • Item: The item that was mined, if any, otherwise null.

Unity Methods

Update

Unity method called once per frame to update the mineable resource.

private void Update();

Usage Examples

Example: Mine a resource

Mineable mineable = GetComponent<Mineable>();
float hitPower = 5.0f;
Item minedItem = mineable.Mine(hitPower);
if (minedItem != null) {
    Debug.Log("Mined item: " + minedItem.Data.name);
} else {
    Debug.Log("No item mined");
}
