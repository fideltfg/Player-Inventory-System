InventorySystemPanel Class

Base class for all inventory panels.

Private Fields

selectedSlotID

The slot that is currently selected in this panel.

private int selectedSlotID;

Public Properties

SelectedSlotID

ID of the slot the player has currently selected.

public int SelectedSlotID { get; set; }

SelectedSlotController

The slot controller of the currently selected slot.

public SlotController SelectedSlotController { get; }

GridLayoutGroup

Holds the GridLayoutGroup component (read-only).

public GridLayoutGroup GridLayoutGroup { get { return GetComponent(); } }

SlotPrefab

The prefab that will be used to generate the slots for this panel.

public GameObject SlotPrefab;

SlotList

Provides a list of all SlotController components of the slots on this panel.

internal List SlotList = new List();

Index

The index of the inventory this panel is displaying.

internal virtual int Index { get; set; }

Public Methods

Build

Called to generate the panel's slots and arrange them on start.

public virtual void Build(int InventoryIndex = 0);

Parameters:

  • InventoryIndex: The index of the inventory to build slots from (default is 0).

Usage Examples

Example: Build the inventory panel

InventorySystemPanel inventoryPanel = GetComponent<InventorySystemPanel>();
inventoryPanel.Build(0);
Debug.Log("Inventory panel built");

Example: Select the next slot

inventoryPanel.SelectedSlotID++;
Debug.Log("Next slot selected");

Example: Get the selected item

Item selectedItem = inventoryPanel.SelectedSlotController.Slot.Item;
if (selectedItem != null) {
    Debug.Log("Selected item: " + selectedItem.Data.name);
} else {
    Debug.Log("No item selected");
}
