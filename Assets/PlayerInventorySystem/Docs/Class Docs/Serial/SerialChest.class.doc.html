SerialChest Class

The SerialChest class represents a serializable chest object in the inventory system.

Private Fields

chestID

The ID of the chest.

[SerializeField] private int chestID;

inventory

The inventory data associated with the chest.

[SerializeField] private SerialInventory inventory;

Constructor

SerialChest

Initializes a new instance of the SerialChest class with the specified item ID and serial transform.

public SerialChest(int itemID, SerialTransform serialTransform);

Parameters:

  • itemID: The ID of the item.
  • serialTransform: The serial transform data.

Public Properties

ChestID

Gets or sets the ID of the chest.

internal int ChestID
{
    get { return chestID; }
    set { chestID = value; }
}

Inventory

Gets or sets the inventory data associated with the chest.

internal SerialInventory Inventory
{
    get { return inventory; }
    set { inventory = value; }
}

Usage Examples

Example: Create a new SerialChest

SerialTransform transformData = new SerialTransform();
// Assume SerialTransform is properly initialized
SerialChest serialChest = new SerialChest(1, transformData);
serialChest.ChestID = 123;
serialChest.Inventory = new SerialInventory();
Debug.Log("SerialChest created with ID: " + serialChest.ChestID);
